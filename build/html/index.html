Welcome to MatD3 Documentation!

This documentation page is designed to guide you through learning about the project and installing it for your personalization of the project.

Introduction

The MatD3 database is designed to store and curate materials data for solid materials. It is part of a NSF-DMREF-funded project (DMR-1729297). This document provides details on the database structure, how to set up a web interface for interacting with the database, and some details on the usage of the website. Currently, there is one live instance of the database, accessible at https://materials.hybrid3.duke.edu/. We also explain how to interface the MatD3 website with the Qresp web application.
